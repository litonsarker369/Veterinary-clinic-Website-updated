# Welfare Veterinary Clinic - Website Redesign

## Project Structure

```
veterinary_clinic_website/
├── index.html          # Main HTML structure
├── styles.css          # All CSS styles
├── script.js           # JavaScript functionality
├── images/
│   └── logo.svg       # Clinic logo (SVG)
└── README.md          # This file
```

## Features

- ✅ Responsive design (Mobile, Tablet, Desktop)
- ✅ WCAG 2.1 AA Accessibility
- ✅ YouTube video embed in hero
- ✅ Instagram Reels section
- ✅ Google Reviews banner
- ✅ Language switcher (EN/AR)
- ✅ Quick action buttons
- ✅ 9 Services with links
- ✅ Smooth animations

## Image Sources

All images use:
- **Unsplash** - High-quality pet photos
- **SVG Logo** - Scalable vector logo

## External Dependencies

- Google Fonts (Cairo, Poppins)
- Font Awesome 6.5.1

## To Use

1. Open `index.html` in a browser
2. Or upload the entire folder to your server
